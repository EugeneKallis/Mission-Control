# Mission Control

A server-hosted service built with Next.js (React frontend + TypeScript backend).

## Quick Start

```bash
just init      # install deps + typecheck
just dev       # start dev server at http://localhost:3000
```

## Commands

See `just --list` or `AGENTS.md` for all commands.

## Structure

- `src/app/` — Next.js App Router (pages + API routes)
- `src/lib/` — shared utilities
- `scripts/` — one-off TypeScript scripts
